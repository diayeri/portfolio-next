## 🚀 Key Achievements

**"디바이스별 최적화된 인터페이스로 기부 문턱을 낮추고, 캠페인의 진정성을 전달하다"**

기존 외주 개발사를 통해 구축되었던 후원 페이지를 서비스 컨셉 변화에 맞춰 전면 개편했습니다. 별도의 외부 공정 없이 **디자인 수정부터 마크업 구현까지 직접 전담**하여 의사결정 단계를 단축하고 프로젝트 리소스를 대폭 최적화했습니다. 기획 의도가 개발 단계에서 왜곡되지 않도록 퍼블리싱을 직접 수행함으로써, 디자인의 완성도를 유지함과 동시에 운영 효율성을 극대화하는 성과를 거두었습니다.

특히 PC와 모바일의 사용 패턴 차이를 반영한 **적응형(Adaptive) 레이아웃**을 기반으로 하되, 각 디바이스 내의 다양한 해상도 변화에도 유연하게 대응하는 **반응형(Responsive) 기법**을 병행하여 구현했습니다. 단순히 레이아웃을 맞추는 것에 그치지 않고, 다양한 해상도에서 컴포넌트 간의 비율이나 가독성이 어색하지 않도록 정교한 검토 과정을 거쳐 사용자에게 일관되고 최적화된 경험을 제공했습니다.

### 1. 시각적 전략: 따뜻함과 가독성의 조화

소액기부라는 취지에 맞춰 사용자가 심리적 부담을 느끼지 않도록 톤앤매너를 설정했습니다.

- **Visual Concept:** 아동들의 밝고 귀여운 사진을 메인으로 활용해 희망적인 분위기를 연출하되, 커버 이미지는 신뢰감을 줄 수 있도록 깊이감 있는 톤을 유지했습니다.
- **Conversion Focus:** 브랜드 아이덴티티를 반영한 Green 컬러를 핵심 CTA에 전략적으로 배치하여, 사용자의 시선 흐름이 자연스럽게 기부 버튼에 머물도록 설계했습니다.

### 2. 기술적 구현: 기기별 최적화 및 직접 퍼블리싱

외부 개발사 없이 직접 코드를 작성하며 디자인 의도를 디테일하게 구현했습니다.

- **PC & Tablet (Vertical One-page):** 넓은 화면을 활용한 세로형 원페이지 스크롤 방식을 채택, 한 화면에 하나의 메시지가 집중되도록 설계하여 몰입감을 높였습니다.
- **Mobile (Horizontal Adaptive UI):** 모바일 기기 사이즈에 맞춰 가로형 스위핑(Swipe) 구조를 적용했습니다.
- **Direct Publishing:** 디자인부터 퍼블리싱까지 직접 담당하여 **커뮤니케이션 비용을 절감하고 전체 제작 일정을 획기적으로 단축**했습니다.

## 📊 Results & Lessons Learned

- **효율적 성과:** 직접 퍼블리싱을 통한 공정 효율화로 재단의 개발 비용을 절감하고 신속하게 캠페인을 런칭했습니다.
- **디자인 임팩트:** 기기별 최적화된 인터랙션과 고정형 버튼 시스템을 도입하여 사용자 편의성을 높였습니다.
- **전문성의 확장:** 디자인과 개발의 경계를 허물고 직접 구현까지 진행하며, 프로젝트 전체 공정을 최적화할 수 있는 실무적 기반을 다졌습니다.
