## 🚀 UX Strategy: 스타와 팬이 함께 만드는 '선한 영향력'의 시각화

### 1. Planning: 팬덤의 '인증' 문화를 시스템화

- **인사이트:** 팬덤은 스타의 이름으로 행한 선행을 온라인에 '인증'하고 공유하며 자부심을 느끼는 특성이 있음.
- **솔루션:** 스타의 사진이 담긴 **기부증서 감사장**과 **미니증서** 리워드를 업계 최초로 기획. 기부 행위를 소장 가치 있는 콘텐츠로 전환하여 자발적 확산을 유도함.

### 2. Implementation: 기술적 완성도를 더한 인터랙티브 UI

- **직접 퍼블리싱을 통한 경험 고도화:** 디자인과 개발의 간극을 줄이기 위해 주요 UI 요소를 직접 구현하여 사용자 편의성을 높였습니다.
  - **Interactive Slider:** 리워드(감사장/포토카드)의 실물을 직관적으로 탐색할 수 있는 슬라이드 기능을 구현해 시각적 몰입감 제공.
  - **Dynamic Accordion:** 캠페인 가이드와 FAQ를 아코디언 UI로 구성하여 정보 밀도를 최적화하고 가독성 확보.
- **선한 영향력의 시각화:** 스타의 품격에 맞는 정교한 디자인 톤앤매너를 유지하면서도, 팬들이 SNS에 인증할 수 있도록 각 플랫폼에 최적화된 레이아웃 설계.

### 3. Impact: 업계 최초가 만든 표준의 힘

- **전환율 극대화:** 직관적인 UI 인터랙션을 통해 별도의 유료 광고 없이 팬덤의 자발적 참여만으로 목표 참여도를 조기 달성.
- **시장 선도 및 확산:** 캠페인 성공 이후, 다수의 NGO 단체들이 본 프로젝트의 모델을 벤치마킹하며 **업계 전체의 팬덤 마케팅 표준**으로 자리 잡음.

### 📊 Professional Growth & Synergy

- **기획과 디자인의 시너지:** 팬덤의 기념일 인증 문화를 기부와 연결하는 기획 단계부터 참여하여, 실제 사용자가 반응하는 실효성 있는 디자인 결과물을 도출했습니다.
- **통합 공정을 통한 효율 확보:** 디자인부터 퍼블리싱까지 직접 진행하여 제작 기간을 단축했으며, PC/Mobile 각 환경에 최적화된 인터랙션을 의도대로 구현했습니다.
- **선한 영향력의 확산:** 디자인을 통해 기부를 즐거운 놀이 문화로 풀어내어, 스타와 팬의 선한 영향력이 SNS상에서 자연스럽게 확산되는 선례를 만들었습니다.
