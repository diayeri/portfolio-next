## 🚀 Key Achievements

아바타 중심의 SNS 서비스 **Stichy**의 웹 앱 반응형 UI와 아티스트용 관리자 페이지(Artist Admin) 개발을 전담하였습니다. 프리랜서로서 클라이언트와 직접 소통하며 전체 일정을 조율하였으며, 페이지 단위의 결과물 전달과 철저한 작업 소요 시간 기록을 통해 프로젝트의 예측 가능성을 높이는 능동적인 프로젝트 관리를 수행했습니다.

특히 앱 기반의 디자인을 웹으로 확장하는 과정에서, 확보된 PC 시안을 바탕으로 **모바일 UI를 직접 디자인하고 구현**하여 기기 환경에 구애받지 않는 일관된 사용자 경험을 구축했습니다.

## 🛠 Challenges & Technical Excellence

### 웹과 앱을 잇는 입체적인 반응형 구조 설계

아바타가 중심이 되는 서비스 특성상, 평면적인 웹에서도 앱과 같은 입체적인 비주얼을 제공하는 것이 중요했습니다. 이를 위해 **SCSS**를 활용하여 레이어 간의 깊이감을 조절하고, 다양한 해상도에서도 아바타와 UI 요소들이 조화롭게 배치되도록 반응형 로직을 설계했습니다.

또한, 반복되는 스타일 요소를 변수화하고 믹스인(Mixin)으로 관리하여 스타일링 작업의 효율을 높였으며, 코드의 중복을 줄여 유지보수가 용이한 구조를 완성했습니다.

### 효율적인 관리자 페이지 구축을 위한 라이브러리 활용

아티스트들이 자신의 아이템을 직접 관리하는 **Artist Admin** 페이지는 빠른 기능 구현과 안정적인 레이아웃 확보가 우선이었습니다. 이를 위해 **Bootstrap** 프레임워크를 도입하여 표준화된 그리드 시스템 위에서 아이템 등록/수정 폼, 파일 업로드 UI 등을 신속하게 구축했습니다. 적응형 UI 방식을 채택하여 관리 업무에 최적화된 화면 구성을 제공함으로써 아티스트들의 사용 편의성을 높였습니다.

### 프리랜서로서의 책임감과 일정 관리 역량

본 프로젝트는 프리랜서 신분으로 참여하여 기획 단계부터 최종 산출물 전달까지 클라이언트와 1:1로 소통하며 진행했습니다. 작업 진행 상황을 투명하게 공유하기 위해 페이지 단위로 피드백 루프를 형성하였고, 매일 작업 소요 시간을 기록하여 남은 일정을 정확히 예측함으로써 클라이언트와의 신뢰 관계를 두텁게 형성했습니다. 이러한 **자기주도적 업무 방식**은 프로젝트가 지연 없이 안정적으로 마무리되는 핵심 동력이 되었습니다.

## 📊 Results & Lessons Learned

프리랜서로서 아바타 기반의 소셜 네트워크 서비스의 웹 퍼블리싱을 담당하며, **비대면 환경에서도 효율적으로 일정을 조율하고 책임감 있게 프로젝트를 완수**하는 협업 프로세스를 익혔습니다. 특히 기획자와 디자이너의 의도를 정확히 파악하여 정해진 기한 내에 결과물을 내놓는 프리랜서만의 기동성과 프로젝트 관리 역량을 키울 수 있었던 소중한 시간이었습니다.

기술적으로는 정형화된 웹 스타일을 벗어나, **게임 UI와 유사한 아바타 시스템을 웹 환경에 구현**하는 새로운 시도에 집중했습니다. 2D 환경에서 3D와 같은 입체감을 주기 위해 레이어의 깊이와 정교한 위치 계산(Positioning) 작업을 수행하며, 서비스 특성에 최적화된 생동감 있는 UI를 구축했습니다. 더불어 Bootstrap 라이브러리를 활용해 관리자 페이지(Admin)의 전체적인 UI 가이드를 정리하며, 서비스의 전면부와 운영 도구를 아우르는 폭넓은 퍼블리싱 경험을 쌓을 수 있었습니다.
